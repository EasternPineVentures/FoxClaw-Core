# Public Hunt Policy

Status: Phase G foundation.

The public FoxClaw Hunt export is a static, sanitized intelligence feed.

Rules:

- Every forecast is labeled `mode=PAPER`.
- Private/internal fields are excluded.
- Losing and resolved forecasts stay visible.
- Superseding receipts are explicit.
- Each export carries an `export_hash`.
- Raw local archives, request payloads, and dossier hashes are not public fields.

The public feed is proof of process, not a profit claim.
