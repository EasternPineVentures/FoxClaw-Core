## Purpose

Planifier turns public intelligence into a personal plan, checklist, journal, and
practice loop. It does not become FoxClaw's scoring authority.

## Current State

Planifier already exists as a built product. It is not being invented here. The
work ahead is integration cleanup, product polish, and a safe contract path from
FoxClaw/CoinFox intelligence into a user-owned planning workflow.

## Planifier May Receive

- `public_intelligence_id`;
- `snapshot_version`;
- public claim summary;
- risk class;
- invalidation guidance;
- time horizon;
- public evidence references;
- public status and authority flags.

## Planifier Must Not Receive

- FoxClaw private engine internals;
- private Grove receipts;
- Apollo private data;
- hidden thresholds;
- live execution authority;
- position sizing commands.

## Product Rule

The user is the variable. Planifier should teach structure, patience,
self-awareness, and executable planning instead of treating any strategy or signal
as one-size-fits-all.

## Resume Trigger

Resume after CoinFox professional cards and public intelligence snapshots exist.
The first useful integration is a plan draft in the existing Planifier app that
references the exact snapshot the user saw.
